Stage 7 ¬∑ Mission Test üß™

Test your GroundCrew.AI agent end-to-end and validate all the capabilities you've built throughout this journey.

Mission Checklist

Follow these steps to thoroughly test your agent's capabilities. Start with general enquiries, then test the 2FA verification flow, and if you completed the bonus stage, test the headshot upload feature.

Access Your Agent
1

In Copilot Studio, navigate to your agent and click "Test" in the top navigation bar. This will open the test chat interface where you can interact with your agent.

üí° Tip: Make sure your agent is published before testing. If you haven't published yet, click "Publish" in the top right corner.
Test General Enquiry Capabilities
2

Start by testing the conversational boosting feature you set up in Stage 4. When the agent greets you, select "General Enquiry" from the options.

Try asking questions like:

  • "What should I bring on my first day?"
  • "What is the dress code?"
  • "What are the working hours?"
  • "Do you have remote work options?"
  • "What benefits are available?"

The agent should use the knowledge base you uploaded in Stage 3 to answer these questions. Verify that the responses are relevant and helpful.

‚úÖ Check: The agent should provide answers based on the knowledge base content. If it doesn't have information, it should politely indicate that.
Test 2FA Identity Verification
3

Now test the 2FA identity verification flow you built in Stage 5. Start a new conversation or return to the conversation start, then select "Verify Email ID".

Follow these steps:

  1. The agent should display an adaptive card asking for your name and email address.
  2. Enter your name and an email address that you can access (use your own email for testing).
  3. Click "Generate OTP üîê".
  4. Check your email inbox for the OTP code.
  5. Enter the OTP code in the adaptive card that appears.
  6. Click "Verify OTP ‚úÖ".
‚úÖ Expected Result: If the OTP is correct, you should see a success message: "Your Email has been successfully verified!" and be returned to the conversation start. If incorrect, you should see "Verification failed" and be prompted to try again.
‚ö†Ô∏è Note: Make sure you're using an email address that you can access. The OTP will be sent to the email you provide in the adaptive card.
Test Headshot Upload (Bonus Feature)
4

If you completed Stage 6 (Crew ID Setup), test the headshot upload functionality. This is an optional bonus feature, so skip this step if you didn't build it.

üéÅ Bonus Feature Testing

To test the headshot upload:

  1. Start a new conversation or return to the conversation start.
  2. Select "Upload headshot" from the options.
  3. The agent should prompt you to upload a file.
  4. Upload an image file (JPG, PNG, etc.) from your device.
  5. Wait for the confirmation message: "Your headshot has been successfully uploaded".
  6. Verify in Power Apps that the image was stored in the UserVerification table's Headshot column.
‚úÖ Verification: Go to make.powerapps.com and check your UserVerification table to confirm the headshot was uploaded correctly.
End-to-End Journey Test
5

Now test the complete user journey as if you were Alex, the new hire:

  1. Start a fresh conversation with the agent.
  2. Ask a general question about onboarding (e.g., "What should I bring on my first day?")
  3. Verify your email using the 2FA flow.
  4. If you built the bonus feature, upload your headshot.
  5. Ask another general question to confirm the agent remembers the context.
üéØ Goal: This end-to-end test simulates a real new hire's experience. The agent should handle each interaction smoothly and guide the user through the pre-onboarding process.
Verify Data in Dataverse
6

Check that all the data is being stored correctly in your Dataverse table:

  1. Go to https://make.powerapps.com
  2. Navigate to your environment and open the UserVerification table.
  3. Verify that:
    • Your name and email are stored correctly
    • The OTP code was generated and stored (if you tested 2FA)
    • The headshot image is present (if you tested the bonus feature)
‚úÖ Success Criteria: All data from your test interactions should be visible in the Dataverse table, confirming that your flows are working correctly.
Mission Complete
7

Congratulations! üéâ You've successfully tested all the capabilities of your GroundCrew.AI agent. Your agent is now ready to help new hires like Alex navigate their pre-onboarding journey.

üöÄ What You've Built: A fully functional pre-boarding agent that can answer questions, verify identities, and collect headshots‚Äîall before Day 1. This creates a smooth, confident landing experience for every new hire.

If you encountered any issues during testing, review the relevant stages to troubleshoot. Otherwise, you're all set! Your GroundCrew.AI agent is ready for deployment.
