Stage 4 ¬∑ Enabling General Enquiries üí¨

Configure the conversation start topic with user options and set up conversational boosting to enable the agent to answer general pre-onboarding questions.

Mission Checklist

Set up the conversation start topic with user options and configure the general enquiry flow using conversational boosting to answer questions from the knowledge base.

Navigate to Topics
1

In Copilot Studio, go to "Topics" on the navigation bar.

Configure Conversation Start
2

Let us allow the agent to orchestrate the conversation from the get-go now. Start by clicking "System" in topics and select "Conversation start".

Delete Message and Add Ask a Question Node
3

Delete the "Message" node and add an "Ask a question" node.

Set Up Welcome Message
4

In the "Message" section, type out any welcome message you would like or just copy and paste this message:

Hey! I am Groundcrew.AI, your personal pre-onboarding assistant, I can help you upload your headshots, verify your Email address and answer any pre-onboarding related questions. Let me know what you would like to do.
Add User Options
5

Click "+ New option" under options for users and add 3 options:

  • Upload headshot
  • Verify Email ID
  • General Enquiry

Copilot Studio would have added 4 "Condition" nodes automatically for you, as seen below. Let us configure just the "General Enquiry" one and leave the rest blank for now.

Configure General Enquiry Flow
6

Click the "+" sign below the "var1 is equal to General Enquiry" node and choose "Topic management", then "Go to another topic", then choose "Conversational boosting".

This allows the agent to use the "Conversational boosting" capability to use the generative AI capabilities and answer questions from users if they choose general enquiries.

Mission Complete
7

Congratulations! You have finished configuring your first topic. You can test your agent's "General enquiry" capabilities.

Try asking questions like "What should I bring on my first day?" or "What is the dress code?" to test the conversational boosting feature.

Your agent can now answer general pre-onboarding questions using the knowledge base you uploaded in Stage 3. Next, we will move to Stage 5 to set up 2FA identity verification.
