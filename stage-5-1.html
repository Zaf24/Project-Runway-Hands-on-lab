Step 5.1 ¬∑ Create Topic üìù

Create a new topic with an adaptive card to collect user email and name for identity verification.

Mission Checklist

Create a new topic from blank, add an adaptive card node to collect user information, and configure it to gather email and name for OTP verification.

Create New Topic
1

On the "Topics" tab in your agent building page, create "+ Add a topic" under custom topics and choose "From blank".

Describe the Topic
2

In the topic builder, describe the topic as you wish, or use this description:

This topic allows user to input their email and verify it using an auto generated OTP
Add Adaptive Card Node
3

Below the trigger node, add an "Ask with adaptive card" node.

Edit Adaptive Card Properties
4

Click on the three dots on the adaptive card node and click "Properties", then select "Edit adaptive card". You will be brought to this page.

Paste Adaptive Card Code
5

On the adaptive card editor, paste this code:

{ "$schema": "http://adaptivecards.io/schemas/adaptive-card.json", "type": "AdaptiveCard", "version": "1.5", "body": [ { "type": "Container", "style": "emphasis", "bleed": true, "items": [ { "type": "TextBlock", "text": "‚ú® Verify Your Identity", "weight": "Bolder", "size": "Large", "color": "Accent", "horizontalAlignment": "Center" }, { "type": "TextBlock", "text": "Enter your details below to generate your OTP üîê", "wrap": true, "spacing": "Small", "horizontalAlignment": "Center" } ] }, { "type": "Container", "style": "default", "items": [ { "type": "TextBlock", "text": "üë§ Full Name", "weight": "Bolder", "spacing": "Medium" }, { "type": "Input.Text", "id": "username", "placeholder": "Your full name" }, { "type": "TextBlock", "text": "üìß Email Address", "weight": "Bolder", "spacing": "Medium" }, { "type": "Input.Text", "id": "useremail", "placeholder": "example@company.com", "style": "Email" } ], "bleed": true } ], "actions": [ { "type": "Action.Submit", "title": "Generate OTP üîê", "data": { "action": "generateOtp" } } ] }
Save and Verify Outputs
6

Click "Save" and exit, and you should see a nice adaptive card with 4 outputs!

Rename the Topic
7

Rename the topic to "Verify email" on the top left instead of "Untitled". Then click "Save".

Congratulations! You just created a topic with an adaptive card. Let's create a flow to generate the OTP next. Move to the next step whenever you are ready!
